# Leo Game in Python

A 2D platformer game inspired by Mario, made with **Python** and **Pygame**. Run, jump, collect coins, and avoid enemies in a fun side-scrolling adventure!

## How to Play
- Use the **arrow keys** to move and jump.  
- avoid enemies and holes

## Setup
- pip install pygame
- python aya.py
  
## Features
- Smooth character movement
- Collectibles and simple enemies
- Add obstacles and enemies
